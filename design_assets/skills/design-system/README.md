# Antigravity Design System Skill

Install per project:

```text
<project-root>/.agent/skills/design-system/
```

Install global:

```text
~/.gemini/antigravity/skills/design-system/
```

Use in Antigravity:

```text
Nutze den design-system Skill und verbessere meine UI.
```

or:

```text
Nutze den design-system Skill und erstelle Design Tokens für mein Projekt.
```
