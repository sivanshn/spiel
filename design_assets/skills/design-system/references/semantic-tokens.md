# Semantic Tokens

Semantic tokens describe purpose.

```css
--color-primary: var(--color-blue-600);
--surface-page: var(--color-gray-950);
--surface-card: rgba(255,255,255,.06);
--text-main: #f8fafc;
--text-muted: #94a3b8;
--border-subtle: rgba(255,255,255,.12);
```

Use semantic tokens in layouts and most components.
